# ChimzyRestaurant OS - User Guide

Welcome to the official manual for **ChimzyRestaurant OS**, the operating system designed for modern multi-branch restaurant empires.

## 1. Central Command Dashboard
**Access**: Login -> Home

The Dashboard is your "God View". It aggregates data from all connected branches in real-time.

![Dashboard](file:///Users/apple/Command Center OS/admin-one-vue-tailwind/assets/dashboard-desktop.png)

### Key Metrics
- **Total Revenue**: Aggregated sales from Cash, POS, and Transfers.
- **Live Orders**: Real-time count of active tickets in the kitchen.
- **System Status**: Health check of your infrastructure (Diesel levels, Freezers).

---

## 2. Live Branch Monitor
**Access**: Sidebar -> Tables / Monitor

This module provides a granular table view of every single branch. Use this to spot specific issues (e.g., "Lekki Branch has low diesel").

![Branch Monitor](file:///Users/apple/Command Center OS/admin-one-vue-tailwind/assets/module-monitor.png)

### Columns
- **Branch**: Name and Location.
- **Sales**: Today's revenue.
- **Diesel/Temp**: Critical infrastructure metrics.
- **Status**: Visual indicator (Green = OK, Red = Critical).

---

## 3. Branch Terminal (Manager Forms)
**Access**: Sidebar -> Forms

Branch managers use this interface to submit end-of-day reports, inventory counts, and manual incident logs.

![Branch Forms](file:///Users/apple/Command Center OS/admin-one-vue-tailwind/assets/module-forms.png)
